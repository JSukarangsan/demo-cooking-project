# Cooking Search Personalization

**Mission:** Cooking | **PM:** Gaelle Sharma | **Created:** 2026-05-05

Personalize search results on NYT Cooking based on user history (saves, cooks, ratings) so discovery feels tailored, not generic. Web-only MVP with A/B experiment by end of Q2 2026.
